Javascript

TYPES OF VALUES:
-Number               var NAME = 0123456789;
-String               var NAME = "WORDS", 'WORDS' and `WORDS`;
-Boolean               var NAME = true/false;
-Typeof               console.log(typeof NAME)

ARITHMETIC OPERATORS:
+               Addition
-               Subtraction
*               Multiplication
/               Division

EQUALITY OPERATORS(GIVES BOOLEAN RESPONSES AS RESULT):
==               Equality, not strict.               1 == 1 // true ... 1 == '1' // true
!=               Inequality, not strict.               1 != 2 // true ... 1 != '2' // true
===               Strict equality, checks both type and value.               1 === 1 // true ... 1 == '1' // false
!==               Strict inequality, checks both type and value.               1 !== 1 // false ... 1 != '1' // true

RELATIONAL OPERATORS:
>               Greater than.               4 > 3 // true
>=               Greater than or equal.               4 >= 3 // true ... 4 >= 4 // true
<               Less than.               13 < 12 // true
<=               Less than or equal to.               3 <= 4 // true ... 3 <= 3 // true

LOGICAL OPERATORS:
&&               AND operator, both sides of operator has to be true.               true && true // true
||               OR operator, either side has to be true.               true || false // true

VARIABLES:
const               assigns a constant that can be any type. It can't be reassigned later in script.               const NAME = TYPE;
let               assigns a variable that can be any type. It can be reassigned later in script.               var NAME = TYPE; ... var = NEWTYPEVALUE;

CONDITIONALS:
if               If sentence, used as a "if condition is met/true, then execute".               if (NAME(value = 10000) > 1000) // true ... { EXECUTE }
if else               If else sentence executes if prior if/if else sentence is not met/true.               if (NAME(value = 1000) > 1000) // false ... if else (NAME(value = 500) > 1000) // false ... if else (NAME(value = 1000) > 1000) // true
else               Else sentence always operates if no other require is met.               if (NAME(value = 1000) > 1000) // false ... else { EXECUTE }

LOOPS:
while               Is a loop that continues till amount "i" is met. Less practical than for-loop.               let i = 0; ... while (i < 5) { EXECUTE; ... i++; }
for               Is a loop that continues till amount "i" is met. Way more practical than while-loop.               for (let i = 0; i < 20; i++) { EXECUTE }
for of               Is loop that will be called as many times as there are elements in array.               for (const name of NAME) { EXECUTE }
forEach               Is a loop that will be called as many times as there are elements in array and give index number as well.               NAME.forEach((name, index) => { console.log(index, name) });

ARRAY:
const NAME = [1, 2, 3, 4, 5];               Creating an area under the NAME.
index's               Locating indexes in arrays are simple. First value is `NAME[0]` and rest is just +1 as counting. It returns the value of the index number.
NAME[0]               Accesses index in array.
reassigning index               You can reassing an index by doing `NAME[0] = 4;`.
.length               Finds the length of an array by `console.log(NAME.length)`.
.push               Adds an element to the end of the array.               NAME.push("6"); ... console.log(NAME); // [1, 2, 3, 4, 5, 6]
.indexof               If you know the value of an index, you can get the number by "NAME.indexof(4); // 3"
for of               Is loop that will be calles as many times as there are elements in array.               for (const name of NAME) { EXECUTE }
forEach               Is a loop that will be called as many times as there are elements in array and give index number as well.               NAME.forEach((name, index) => { console.log(index, name) });

FUNCTIONS:
Creating a function:               function name (PARAMETERS) { EXECUTE ... return RESULT };
Creating an arrow-function:               const NAME = (PARAMETERS) => { EXECUTE };

Math.random()               Will give a random number between 0 and 1.               Math.randoom() // 0-1
STRING.trim()               Will remove excess whitespaces.               STRING.trim() // Removes excess whitespaces
console.log()               Used to log out content.               console.log("TEXT") // Logs out "TEXT"
Math.max               Will return the highest number of numbers.               Math.max(3, 5) // 5
STRING.length               Will return the length of the string. // Number
STRING.toUpperCase()               Will return all string uppercase. // helLo -> HELLO
STRING.toLowerCase()               Will return all string lowercase. // HElLO -> hello
STRING.charAt(0)               Will return character at index. // Letter
STRING.charCodeAt(0)               Will return the ASCII code of the index. // Number
STRING.slice(0, 0)               Will return part that is specified by index. // String
STRING.substring(0, 0)               Will return part that is specified by index. // String
STRING.substr(0, 0)               Will return part that is specified by index. // String from index + second number
STRING.indexOf("")               Will return the index value. // Number
STRING.lastIndexOf("")               Will return the index value of last match. // Number
STRING.includes("")               Will return boolean whether it contains string. // true/false
STRING.startsWith("")               Will return boolean whether it starts with string. // true/false
STRING.endsWith("")               Will return boolean whether it ends with string. // true/false

CALLBACK FUNCTIONS(A FUNCTION INSIDE OF ANOTHER FUNCTION):
setTimeout()               Is made to run a function after a specific amount of time.               setTimeout(function() {}, TIME); // TIME in milliseconds, after set time the function will execute.
setInterval()               Will execute at a set interval.               setInterval(function() {}, TIME); // Will execute callback function every set TIME/interval.

OBJECTS:
Creating an object:               This is the so-called Javascript Objects structure:               const NAME = { key1: "VALUE", key2: "VALUE", key3: "VALUE" };
Accessing data from JS object:               This will find the specific value of a key in our object:               NAME.key OR NAME.[key]
Changing data using key:               We can change the value of a key using the key:               NAME.key = NEWVALUE; // Updates value to NEWVALUE
Array of objects:               Multiple objects can be stored into 1 single array.               const NAME = [{ key1: "VALUE", key2: "VALUE", key3: "VALUE", },{ key1: "VALUE", key2: "VALUE", key3: "VALUE", }]

DOM(Document Object Model):
document.querySelector("")               Selects an element in the HTML.               const NAME = document.querySelector("");
document.querySelectorAll("")               Selects all elements of the same kind in the HTML.               const NAME = document.querySelectorAll(""); // NOTE: will return as a nodelist.
Array.from(document.querySelectorAll(""));               Turns nodelist into array.
.textContent               Will give the text content inside the selected HTML element.               const NAME document.querySelector("").textContent; // Text content inside element
.getAttribute               Will give the requested attribute.               const NAME = document.querySelector("").getAttribute(""); // Gets wanted attribute
.innerText               Will retrieve text that is visible to the user only. If text is hidden it won't show.
.className               Used to finde class and edit it.               NAME.className = "VALUE";
.setAttribute               Used to update or set an attribute.               NAME.setAttribute("ATTRIBUTE","VALUE")
.insertAdjacentHTML               Will place a element into the HTML.               NAME.insertAdjacentHTML('afterend', `<.h2>TEXT<./h2>`)
.addEventListener()               Will give the HTML element a detector/activator option.               NAME.addEventListener("click", NAME )

FETCH:
Fetch is used to send requests to get some data from an api. Then it parses it so its in JSON format so we can work with it:
fetch('URL')
               .then(response => response.json())
               .then(dataName => {
                              console.log(dataName);
               });

CHART.JS:
Chart.JS is a method to show data in HTML using javascript. To use it we need to querySelect the element we wish to do as a chart:
const NAME = document.querySelector('#chart');

Then afterwards we create the chart in javascript:
const chart = new Chart(NAME, {
               type: 'bar',
               data: {
                              labels: [1, 2, 3, 4, 5],
                              datasets: [{
                                             data: [1, ,2 ,3, 4, 5],
                              }]
               }
});
